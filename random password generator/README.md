# Password Generator

A simple Python project that generates strong random passwords using letters, numbers, and symbols.

## Features
- Generate random passwords
- Custom password length

## Technologies Used
- Python

## Run the Project

```bash
python password_generator.py
```

## Example

```bash
Enter password length: 10

Generated Password: A@7kL#2pQ!
```
